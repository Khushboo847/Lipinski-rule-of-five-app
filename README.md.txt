# Lipinski Rule of Five Filter

Interactive web app to evaluate drug-likeness using Lipinski’s Rule of Five.

* Features
- SMILES input
- Molecular property calculation
- Rule-of-five violation detection
- Molecule visualization

* Properties Calculated
- Molecular Weight
- LogP
- H-bond Donors
- H-bond Acceptors

* Run Locally
pip install -r requirements.txt  
streamlit run lipinski_app.py
